# Vibe Coding MVP - 학습 플랫폼 프로토타입

## 📌 프로젝트 개요

**Vibe Coding**은 "딸깍→통제→검증→자기화→포트폴리오" 철학을 기반으로 한 코딩 학습 플랫폼 MVP입니다.

초등학생도 쉽게 사용할 수 있도록 설계되었으며, 다음 핵심 원칙을 따릅니다:

- **즉시 실행**: 복잡한 설정 없이 바로 시작
- **테스트 중심**: 감이 아닌 명확한 검증으로 성공 판정
- **튜터 지원**: 정답 제공이 아닌 질문/힌트/부분 예시로 학습 유도
- **포트폴리오 자동 생성**: 완성한 작품이 자동으로 저장

---

## 🎯 주요 기능

### 1. 홈 - 오늘의 미션 (index.html)
- 미션 소개 동영상 (10~15초)
- 간단한 브리핑 카드 (산출물, 성공 조건, 완성 후 혜택)
- 바로 만들기 / 샘플 보기 버튼
- 최근 완성한 작품 미리보기

### 2. 워크스페이스 (workspace.html)
**좌측 (30%): 플레이 영역**
- 게임 실행/리셋/전체화면
- 실시간 점수 및 상태 표시

**우측 (70%): 3개 탭**
- **테스트 탭** (기본): 3개 테스트 체크리스트
  - 각 테스트마다 "상황/입력" → "기대 결과" 명시
  - 실행 → 통과/실패 수동 판정
  - 실패시 디버그 섹션 자동 노출 → 튜터로 연결
  
- **튜터 탭**: AI 도움 시스템 (3단계)
  - Level 1: 질문으로 생각 유도
  - Level 2: 힌트와 확인 방법 제공
  - Level 3: 부분 예시 (최대 2줄) + 이유 + 다음 행동
  
- **도움말 탭**: 동영상 재시청 및 핵심 팁

**하단: 코드 편집 영역 (토글 가능)** 🆕
- **▼ 코드 보기** 버튼으로 펼치기/접기
- **좌측 (20%)**: 파일 트리
  - main.js, game.js, style.css
  - 파일 클릭으로 전환
  - 새 파일 생성 버튼
- **중앙 (60%)**: 코드 에디터
  - 기본: 읽기 전용 (초보자 보호)
  - 체크박스: "직접 수정 (고급)" 모드
  - 문법 하이라이팅 (다크 테마)
- **우측 (20%)**: 콘솔/로그
  - 실행 결과 실시간 표시
  - console.log() 출력
  - 오류 메시지 (빨간색)
  - 지우기 버튼

**상단 바**
- 통과 상태 (0/3)
- 저장 상태
- 오류 표시
- 제출 버튼 (3개 테스트 통과시 활성화)

### 3. 마이페이지 (mypage.html)
- 사용자 프로필 및 통계
- 완성한 작품 카드 리스트
  - 작품 보기
  - 공유하기
- 최근 활동 로그

## 🚀 시작하기

### 실행 방법
1. `index.html` 파일을 브라우저에서 열기
2. "바로 만들기" 버튼 클릭
3. 워크스페이스에서:
   - **"▼ 코드 보기"** 버튼 클릭해서 코드 영역 펼치기
   - main.js 파일에서 코드 확인
   - "직접 수정 (고급)" 체크박스로 편집 모드 활성화
   - 코드 수정 → "▶ 코드 실행" 버튼
   - 테스트 1~3 실행하고 "✓ 통과" 클릭
4. **3개 통과하면 제출 버튼 활성화**
5. **마이페이지**에서 포트폴리오 확인

### 파일 구조
```
vibe-coding-mvp/
├── index.html          # 홈 (미션 소개)
├── workspace.html      # 워크스페이스 (메인 작업 공간)
├── mypage.html         # 마이페이지 (포트폴리오)
├── style.css           # 공통 스타일
├── home.js            # 홈 로직
├── workspace.js       # 워크스페이스 로직
├── mypage.js          # 마이페이지 로직
└── README.md          # 프로젝트 문서
```

## 🎯 설계 철학

### 테스트 중심 학습
- "감"이 아닌 **명확한 테스트 통과**로 성공 판정
- 튜터는 판정하지 않고, **테스트가 판정**

### 튜터의 역할
- ❌ 정답 제공
- ✅ 질문으로 생각 유도
- ✅ 힌트로 방향 제시
- ✅ 부분 예시 (최대 2줄)로 학습 보조

### 45분 완성 보장
- 메인 미션: 45분 내 완성 가능하도록 설계
- 확장 미션: 15분 추가 (선택)
- 느린 학생도 제출만으로 성공 경험

### 포트폴리오 중심
- 완성한 작품은 **자동으로 포트폴리오에 저장**
- 의도/조건/테스트가 **자동으로 요약**
- 학부모/교수자에게 **가시성 제공**

## 💡 사용 시나리오

### 일반적인 학습 흐름
```
1. 홈 → "바로 만들기" 클릭
2. 동영상 시청 (10~15초)
3. 워크스페이스 진입
4. 게임 실행 → 테스트 1 실행 → 통과 클릭
5. 테스트 2 실행 → 실패 클릭 → 디버그 섹션 노출
6. 튜터 탭 → Level 2 선택 → 힌트 확인
7. 테스트로 돌아가기 → 다시 실행 → 통과
8. 테스트 3 통과
9. 제출 버튼 활성화 → 제출
10. 마이페이지로 이동 → 포트폴리오 확인
```
